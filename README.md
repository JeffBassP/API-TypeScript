### Criando uma API com TypeScript

  ## Middlewares utilizados 
  * compression
  * Express
  * Cors 
  * Helmet 
  * http-status-code 

  ## Para utilizar a API:
    - execute `npm install`
    - execute `npm run dev`
    - use o POSTMAN com o endereco de http://localhost/users/
  
  ## API com funçoes de CRUD
    - Get
    - Post 
    - Put
    - Delete

  *** Futuramente será implementado o Swagger, se possivel ***
